# Values And Anti Patterns（融会贯通版）

## 维度结论

价值观层面呈现出一个清晰取向：

1. **主体性优先**
- 人是目的，不是手段。
- 做选择时优先保证主观能动性，不把自己变成被动执行体。

2. **长期一致性优先**
- 看重“是否成为想成为的人”，不只看短期结果。
- 能持续做下去，比短期漂亮更重要。

3. **关系与情感不是噪声**
- 情感与关系被视为长期系统中的关键变量。
- 稳定的人际与心理状态，是执行质量的前提条件。

## 反模式库

1. **多线程焦虑**：同时维护多个主线，最后每条都浅。  
2. **伪忙碌**：做大量低价值任务获取短期安慰。  
3. **完美主义内耗**：因追求一次到位而长期不动。  
4. **外部评价驱动**：用他人标准替代自我主线。

## 边界原则

- 不透支健康去换短期进度。  
- 不在价值冲突明显时硬做“理性最优”。  
- 不以“成熟”为名长期压制真实目标。
